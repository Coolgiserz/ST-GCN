# STGCN-Collections
This repos intergrate the popular models for spatail-temporal prediction, such as:
- ST-GCN
- Graph-WaveNet


 ## Requirements
  
  * PyTorch
  * NumPy
  * Matplotlib
  
 ## Example Dataset
  
- METR-LA
- PEMS-Bay

## Features
- tensorboard support
- multi metrics for evalution
- logger
